นำเข้าจากไฟล์ Excel เดิม

เลือกไฟล์ HOBI_ระบบบัญชีร้าน.xlsx เพื่อดึงข้อมูล Orders / Order_Items / Purchases / Menu เข้ามาในแอพ (ข้อมูลจะถูกเพิ่มเข้าไป ไม่ลบของเดิมในแอพ)

ส่งออกเป็น Excel

ดาวน์โหลดข้อมูลทั้งหมดในแอพตอนนี้เป็นไฟล์ .xlsx (Orders / Order_Items / Purchases / Menu) — ไฟล์นี้เป็นข้อมูลล้วนๆ ไม่มีสูตร/แดชบอร์ดแบบไฟล์เดิม เพราะแดชบอร์ดอยู่ในแอพนี้แล้ว

ลบข้อมูลทั้งหมด

ระบบจะดาวน์โหลดไฟล์ Excel สำรองข้อมูลปัจจุบันให้อัตโนมัติก่อนลบทุกครั้ง และต้องพิมพ์ยืนยันให้ตรงก่อนปุ่มลบถึงจะกดได้ เพื่อกันเผลอกดโดน

☁️ ซิงค์ข้อมูลข้ามเครื่อง (Cloud Sync)

เชื่อมต่อ Firebase (ฟรี) เพื่อให้กรอกข้อมูลจากมือถือหรือคอมก็ได้ ข้อมูล sync กันอัตโนมัติแบบเรียลไทม์ ต้องตั้งค่าโปรเจกต์ Firebase ก่อนครั้งเดียว:

1. เข้า console.firebase.google.com → สร้างโปรเจกต์ใหม่ (ฟรี)
2. เมนูซ้าย "Build" → Firestore Database → Create database → เลือกโหมดใดก็ได้ (ตั้งค่า Rules เพิ่มด้านล่าง)
3. เมนูซ้าย "Build" → Authentication → Get started → เปิดใช้ผู้ให้บริการ "Google"
4. Authentication → Settings → Authorized domains → เพิ่มโดเมนเว็บที่ใช้เปิดแอพนี้ (เช่น ยูสเซอร์.github.io)
5. ไปที่ ⚙️ Project settings → เลื่อนลงหา "Your apps" → กด "Add app" เลือกไอคอนเว็บ (</>) → คัดลอกโค้ด firebaseConfig ทั้งก้อนมาวางด้านล่างนี้
6. ที่ Firestore → แท็บ Rules วางกฎนี้แล้ว Publish:

ยังไม่ได้เชื่อมต่อ Cloud Sync
💡 ครั้งแรกหลัง sign in แล้ว ถ้ามีข้อมูลเดิมอยู่ในเครื่องนี้ (จากก่อนเปิด Cloud Sync) ให้กดปุ่มด้านล่างเพื่ออัพโหลดข้อมูลเดิมขึ้นคลาวด์ครั้งเดียว
