Help & How-To ▾

รวมคำอธิบายทุกจุดในแอพไว้ที่เดียว อ่านรอบเดียวจบ — ปุ่ม (i) เล็กๆ ตามหน้าต่างๆ ก็ยังกดดูซ้ำได้เหมือนเดิมถ้าลืม

Upload (Home)

ส่งรูปแคปออเดอร์/ใบเสร็จไปให้ Claude อ่านในแชทปกติ (ไม่มีค่าใช้จ่ายเพิ่ม) แล้วก็อปโค้ด JSON ที่ได้มาวางตรงนี้ กด "Import" — ระบบจะพักข้อมูลไว้ให้ตรวจสอบทีละรายการก่อนบันทึกจริง (รองรับทั้งออเดอร์และรายการซื้อของในโค้ดเดียวกัน)

Imported data to be confirmed (Home)

หลังนำเข้าข้อมูลจาก Claude Chat แล้ว ตรวจสอบรายการด้านล่างนี้ก่อน กด "ยืนยัน" เพื่อบันทึกจริงเข้าไปในลิสต์ของหน้าออเดอร์/ซื้อของ/การเงินทันที หรือกด "ลบ" ถ้าไม่ต้องการรายการนี้ (แก้ไขรายละเอียดเพิ่มเติมได้ทีหลังจากลิสต์ในแต่ละหน้า)

แคมเปญ 60/40 (Orders)

ค่าคอมมิชชั่นคำนวณต่างจากปกติ และยอดขายนี้จะไม่โชว์ใน Grab Earnings (หน้า Finance โชว์แต่ยอดหักลบ) — ระบบจะจดไว้ในหมายเหตุให้อัตโนมัติ

ค่าธรรมเนียมแพลตฟอร์มต่อออเดอร์ (Orders)

กรอกตรงนี้ได้เลยถ้าเช็คค่าคอม/ค่าโฆษณาต่อออเดอร์ได้ (เช่น Grab ดูได้จากหน้า Transactions) — ถ้าไม่สะดวกดูรายออเดอร์ (เช่น LineMan ที่หน้าการเงินให้แค่ยอดรวมทั้งวัน) ปล่อยเป็น 0 ไว้แล้วไปบันทึกรวมเป็นยอดรายวันที่หน้าการเงินแทนได้เลย ระบบจะไม่นับซ้ำ — ถ้าวันนั้น+ช่องทางนั้นมีบันทึกรวมรายวันแล้ว จะใช้ยอดรวมรายวันแทนค่าที่กรอกต่อออเดอร์ให้อัตโนมัติตอนคำนวณกำไร

ชื่อไอเทม autocomplete + ส่วนต่างท้ายบิล (Purchases)

พิมพ์ชื่อไอเทมในฟอร์มซื้อของไม่กี่ตัวอักษร ระบบจะโชว์ชื่อที่เคยกรอกไว้แล้วในหมวดหมู่เดียวกันให้เลือกซ้ำ (กดเลือกจากลิสต์แทนพิมพ์ใหม่ ชื่อจะตรงกันทุกครั้งแม้ซื้อจากคนละร้าน) แต่ละแถวจะโชว์ยอดรวม (จำนวน×ราคา) ให้อัตโนมัติด้วย — ถ้ายอดที่จ่ายจริงท้ายบิลไม่ตรง (มีส่วนลด/ค่าส่งเพิ่ม) ใส่ผลต่างในช่อง "ส่วนต่างท้ายบิล" ก้อนเดียว ระบบจะกระจายเฉลี่ยลงแต่ละไอเทมให้เอง ไม่ต้องคำนวณเอง

Cross-Device Sync (Cloud Sync)

เชื่อมต่อ Firebase (ฟรี) เพื่อให้กรอกข้อมูลจากมือถือหรือคอมก็ได้ ข้อมูล sync กันอัตโนมัติแบบเรียลไทม์ ต้องตั้งค่าโปรเจกต์ Firebase ก่อนครั้งเดียว:

1. เข้า console.firebase.google.com → สร้างโปรเจกต์ใหม่ (ฟรี)
2. เมนูซ้าย "Build" → Firestore Database → Create database → เลือกโหมดใดก็ได้ (ตั้งค่า Rules เพิ่มด้านล่าง)
3. เมนูซ้าย "Build" → Authentication → Get started → เปิดใช้ผู้ให้บริการ "Google"
4. Authentication → Settings → Authorized domains → เพิ่มโดเมนเว็บที่ใช้เปิดแอพนี้ (เช่น ยูสเซอร์.github.io)
5. ไปที่ Project settings → เลื่อนลงหา "Your apps" → กด "Add app" เลือกไอคอนเว็บ (</>) → คัดลอกโค้ด firebaseConfig ทั้งก้อนมาวางด้านล่างนี้
6. ที่ Firestore → แท็บ Rules วางกฎนี้แล้ว Publish:

ยังไม่ได้เชื่อมต่อ Cloud Sync
ครั้งแรกหลัง sign in แล้ว ถ้ามีข้อมูลเดิมอยู่ในเครื่องนี้ (จากก่อนเปิด Cloud Sync) ให้กดปุ่มด้านล่างเพื่ออัพโหลดข้อมูลเดิมขึ้นคลาวด์ครั้งเดียว
